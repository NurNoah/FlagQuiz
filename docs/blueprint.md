# **App Name**: Flaggen Quiz

## Core Features:

- Flag Display: Display a randomly selected country flag.
- Country Guess Input: Provide an input field for users to guess the country.
- Answer Validation: Check the user's answer against the correct country name and provide immediate feedback.
- Score Tracking: Keep track of the user's score during the session.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for a fresh and inviting feel.
- Background color: Light gray (#F0F8FF) for a clean and unobtrusive backdrop.
- Accent color: Royal blue (#4169E1) to highlight correct answers and interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, suitable for headlines or body text
- Use simple, clear icons to represent options or provide hints.
- Mobile-first design with a responsive layout to ensure optimal viewing on different screen sizes.
- Subtle animations on correct answers to enhance user experience.
